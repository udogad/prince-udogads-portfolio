# 🌌 Quantum Portfolio: Senior Systems Architect

A high-fidelity, autonomous portfolio experience engineered for the future. This project showcases the intersection of **Scalable Web Architectures**, **Cross-Platform Mobile Ecosystems**, and **Generative AI Integration**.

## 💎 Design Philosophy
The "Quantum" design focuses on high-contrast, cinematic aesthetics using fluid typography and glassmorphism. It is built to stand the test of time by utilizing modular components and a headless content architecture.

## 🛠 Engineering Core
- **Intelligence Layer**: Integrated **Gemini 3 Flash** via `@google/genai` for a real-time conversational assistant that understands the developer's entire project history.
- **Sentinel Protocol**: A proprietary visitor monitoring system that uses **Geolocation APIs** and **Webhooks** to track and verify engagement in real-time.
- **Visual Analytics**: Interactive **Skills Radar** built with Recharts to visualize engineering proficiency across the three primary pillars (Web, Mobile, AI).
- **Architecture**: React 19, Tailwind CSS, and Lucide for a lightweight, dependency-conscious frontend.

## 🚀 Deployment & Configuration

### Prerequisites
- An API Key from [Google AI Studio](https://aistudio.google.com/).
- (Optional) A Discord/Slack Webhook URL for the Sentinel Protocol.

### Environment Setup
The application relies on the following environment variable:
```env
API_KEY=your_gemini_api_key_here
```

### Key Modules
- `constants.tsx`: The source of truth. Edit this file to update projects, skills, and contact info.
- `services/geminiService.ts`: Manages the conversational logic and function calling (tool use).
- `components/SentinelPulse.tsx`: Handles the real-time engagement matrix and visitor logging.
